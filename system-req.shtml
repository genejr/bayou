System Requirements

!nService is built to run on multiple platforms. Regardless of the size and focus of your organization's IT department, !nService can be tailored to support your needs.

Operating System - Server

Red Hat Enterprise Linux, Ubuntu or CentOS.

Operating System - Client

Windows (all versions), Mac OS X, iOS, Android & Linux.

Browsers Supported

Chrome, Firefox, IE 8+ & Safari.

Tablets and Devices

iPad, iPhone, MS Surface, Android tablets & Android smartphones.

Memory - Server

Minimum 4GB, Recommended 8GB

Memory - Client

Minimum 2GB, Recommended 4GB

Harddisk - Server

1.8GB for installation, min 6TB Free disk space recommended for 5 years of Case retention
